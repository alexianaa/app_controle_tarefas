# Controle de tarefas

Para rodar o projeto:

1. npm install
2. npm run dev
3. em outro terminal rodar php artisan serve

Para ter tarefas e usuários pré feitos você pode rodar

> php artisan db:seed

E usar o usuário padrão sendo o email "test@example.com" com senha "password"
